## Working Version ##
** Pull Requests are welcome by all. **

You can install it locally by following these instructions:

1. `git clone https://github.com/DZuz14/React-Image-Viewer.git`
2. `npm install`
3. `npm run dev`
4. 'npm run build'
5. 'npm start'
6. Visit `localhost:8080` in your browser of choice.
